# MaiPDF

**官网 / Website:** [maipdf.com](https://maipdf.com) · **App DRM:** [drm.maipdf.com](https://drm.maipdf.com)

MaiPDF 帮助你在分享 PDF 时保留控制权——防止随意复制、追踪阅读记录，并在需要时随时撤销访问。

MaiPDF helps you share PDFs with control — restrict copying, track opens, and revoke access when needed.

---

## 两种使用方式 · Two Ways to Use MaiPDF

### 1. 在线版 · Online Sharing

适合快速、受控的浏览器分享。上传 PDF，设置规则，生成链接或二维码。

- 过期时间与打开次数限制
- 动态水印与阅读记录
- 读者无需安装 App
- 控制中心随时调整权限、替换文件

**开始使用 →** [maipdf.com](https://maipdf.com/)

### 2. App DRM 版 · App DRM

适合机密文件。将 PDF 打包为 `.maipdf`，在专用 App 中打开，获得系统级防护。

- 防截图、防录屏（各平台策略不同）
- 邮箱授权与设备绑定
- 按页水印与追踪码
- 随时暂停或撤销访问

**了解更多 →** [drm.maipdf.com](https://drm.maipdf.com/)

---

## 如何选择 · Which One to Choose?

| 场景 | 推荐 |
|------|------|
| 快速发链接、二维码，读者用浏览器打开 | **在线版** |
| 文件本身需要交付（邮件、网盘、U 盘） | **App DRM** |
| 需要防截图、设备限制、强撤销能力 | **App DRM** |
| 不想让读者安装任何东西 | **在线版** |
