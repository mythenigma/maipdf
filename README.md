# MaiPDF

**Website:** [maipdf.com](https://maipdf.com) · **App DRM:** [drm.maipdf.com](https://drm.maipdf.com)

MaiPDF helps you share PDFs with control — restrict copying, track opens, and revoke access when needed.

---

## Two Ways to Use MaiPDF

### 1. Online Sharing

Best for quick, controlled browser sharing. Upload a PDF, set rules, and generate a link or QR code.

- Expiry dates and open limits
- Dynamic watermark and read tracking
- No app required for readers
- Control center to adjust permissions and replace files later

**Get started →** [maipdf.com](https://maipdf.com/)

### 2. App DRM

Best for confidential files. Package PDFs as `.maipdf` and open them in a dedicated app with OS-level protection.

- Anti-screenshot and screen-recording controls (varies by platform)
- Email authorization and device binding
- Per-page watermarks and trace codes
- Pause or revoke access anytime

**Learn more →** [drm.maipdf.com](https://drm.maipdf.com/)

---

## Which One to Choose?

| Scenario | Recommended |
|----------|-------------|
| Send a link or QR; readers open in the browser | **Online Sharing** |
| Deliver the file itself (email, cloud, USB) | **App DRM** |
| Need anti-screenshot, device limits, strong revoke | **App DRM** |
| Readers should not install anything | **Online Sharing** |
